## 테스트 전략
* **시나리오 기반 테스트** : 대기실 → 던전 입장 → 패턴 발생 → 클리어 로 이어지는 전체 사이클을 검증합니다.
* **기능 테스트** : 클리어시 보상 획득, 아이템 교환을 검증합니다.
* **사용성 및 UX 테스트** : 신규 콘텐츠 업데이트 데빌 오브 카오스의 기믹 테스트.
