## 👤 도메인 지식 및 숙련도
**"해당 게임사(KOG)의 게임을 깊이 있게 플레이하며 시스템 로직과 유저 패턴을 파악하고 있습니다."**

* **플레이 목록**
* **엘소드** : 2008년 체험아이디로 시작하여 현재까지 플레이하고있으며 현재 모든 콘텐츠를 클리어 경험이 있으며
영웅대전, 팝업스토어, 엘스타, 오케스트라 등의 오프라인 이벤트도 참여하였습니다.
* **그랜드체이스** : 2009년 ~ 2012년 "아신" 캐릭터 출시까지의 플레이 경험을 가지고있으며 캐릭터 영입 퀘스트, 전직 퀘스트, 펫 조합식 등
다양한 플레이경험을 보유하고 있습니다.
* **커츠펠** : 출시후 플레이해보았으며 기존과 다른 PVP와 던전의 요소를 경험하였습니다.


## 🔍 프로젝트 개요
* **목적**: 엘소드 신규 콘텐츠 헤니르 : 데빌 오브 카오스, 하이엔트로피의 던전 입장 ~ 클리어 까지의 시나리오를 테스트하며 정상적으로 작동하고있는지를 검증하기위한 테스트입니다.

## 테스트 전략
* **시나리오 기반 테스트** : 마을 → 던전 입장 → 패턴 발생 / 보스 처치 → 클리어 로 이어지는 전체 사이클을 검증합니다.
* **기능 테스트** : 클리어시 보상 획득, 아이템 교환을 검증합니다.
* **경계값 분석 테스트** : 입장전투력 미달, 충족(적정 전투력 -5% 이내), 충족 상태의 던전입장을 검증합니다.
* **사용성 및 UX 테스트** : 신규 콘텐츠 업데이트 데빌 오브 카오스의 기믹 테스트.
