[TC-010] 데빌 오브 카오스 심연의 숭배자 줄넘기 패턴
Status: 🟢 Pass 

개요 (Summary)
헤니르 데빌 오브 카오스의 심연 지역 매몰된 성전 1페이즈 줄넘기 패턴의 정상적 작동 테스트

사전 조건 (Pre-conditions)
데빌 오브 카오스 입장 가능캐릭터

테스트 단계 (Test Steps)
| Step | Action | Test Data | Expected Result | Actual Result |
|:---:|---|---|---|:---:|
| 1 | - | 던전입장 or 이전 스테이지 클리어 | 보스 등장 컷신 연출 출력 | 🟢 Pass |
| 2 | 컷신 진행중 | 키보드 ESC 입력 | 컷신 스킵 | 🟢 Pass |
| 3 | 컷신 진행중 | 대기 | 보스 등장 영상 종료 후 맵 중앙 발판에 스폰 | 🟢 Pass |
| 4 | - | 보스행동 확인 | 보스가 맵뒤로 이동하며 맵 상단, 하단에 안개 생성 | 🟢 Pass |
| 5 | 패턴진행 | 오브젝트 확인 | 중앙발판 양끝단에 자색 그림자 생성, 각 그림자 사이 레이저 곡선 연결 | 🟢 Pass |
| 6 | - | 패턴진행 | 연결된 레이저가 가속, 감속을 반복하며 회전 약 25초간 진행 | 🟢 Pass |
| 7 | - | 레이저 피격 | 피격시 체력의 약 30% 데미지 | 🟢 Pass |
| 8 | - | 체력 10% 미만 | 피격시 캐릭터의 사망 | 🔴 Fail |
| 9 | - | 맵 상단, 하단의 그림자 접촉 | 맵 중앙으로 텔레포트 후 기절 디버프 2초 부여 | 🟢 Pass |
| 10 | - | 레이저를 점프로 끝까지 회피 | 생존후 CLEAR 메시지 출력 | 🟢 Pass |
