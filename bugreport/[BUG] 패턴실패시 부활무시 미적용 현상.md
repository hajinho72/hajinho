[BUG] 패턴실패시 부활무시 미적용 현상
버그 요약: 어떤 버그인지 한 문장으로 설명해주세요.

💻 테스트 환경 

OS/버전: Windows 11

대상 버전: L.260129.1_5

📋 재현 단계 

1. 서펜티움 지역 오케스트라 홀 싱글모드 진입
2. 스킬 사용으로 문양 활성화 하는 패턴 진입
3. 테스트 목적으로 2개다 비활성화 후 패턴 실패
4. 부활칭호 효과를 받으며 부활

✅ 기대 결과 

메인 패턴 실패시 부활무시 형태의 즉사 공격을 실행

❌ 실제 결과 

다른 메인 패턴 실패결과가 부활무시상태 즉사 판정이지만 해당 패턴은 부활칭호가 적용되어 
패턴이 실패하여도 부활하는 현상
