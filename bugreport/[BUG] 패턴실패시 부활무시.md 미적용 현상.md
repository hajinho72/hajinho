 [BUG] 레이드 패턴 실패 시 캐릭터가 자동 부활하는 현상

버그 요약: 서펜티움 레이드에서 즉사급 메인 패턴 실패 시 캐릭터가 사망하지 않고 부활 칭호 효과로 즉시 부활하여 공략이 무력화됩니다.

💻 테스트 환경 (Environment)

OS/버전: Windows 11

대상 버전: L.260129.1_5

📋 재현 단계 (Steps to Reproduce)

서펜티움 지역 오케스트라 홀 싱글 모드 진입

스킬 사용으로 문양을 활성화하는 메인 패턴 진입

테스트 목적으로 문양 2개를 비활성화 상태로 유지하여 패턴 실패 유도

부활 칭호 효과를 받으며 자동으로 부활하는 현상 확인

✅ 기대 결과 (Expected Result)

메인 패턴 실패 시 '부활 무시' 형태의 즉사 공격을 실행하여 캐릭터가 반드시 사망해야 함.

❌ 실제 결과 (Actual Result)

다른 메인 패턴 실패 결과는 '부활 무시' 상태의 즉사 판정이 정상 적용되나, 해당 패턴에 한해서만 부활 칭호가 적용되어 패턴 실패 후에도 전투가 지속되는 현상 발생.
